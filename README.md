# Principles-of-Data-Science
Data Analysis on AutoTrader dataset
This project explores the AutoTrader UK vehicle dataset, focusing on data understanding, cleaning, feature engineering, and exploratory analysis.
It forms the EDA and data‑science foundations for later machine‑learning work on the same dataset.

The analysis covers:

Understanding feature types

Handling missing values and outliers

Creating new engineered features

Subsetting and grouping

Quantitative–quantitative, quantitative–categorical, and categorical–categorical analysis

Visual exploration of trends in price, mileage, age, fuel type, body type, and more

The project is implemented in Python (Pandas, NumPy, Matplotlib, Seaborn) and was developed in Google Colab.
